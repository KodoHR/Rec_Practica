Cómo armar tu Proyecto en ChatGPT

  1. Entrá a chatgpt.com y en la barra lateral izquierda tocá "Proyectos".
  2. Creá uno nuevo y nombralo "Reclutadora Sr. Kodo — [tu nombre]".
  3. Abrí "Instrucciones del proyecto" y pegá el texto de abajo (reemplazando [TU NOMBRE]).
  4. Listo. Cada chat nuevo que abras dentro de este proyecto ya arranca con ese contexto.

Cómo armar tu Proyecto en Claude

  1. Entrá a claude.ai y en la barra lateral izquierda tocá "Proyectos" → "Crear proyecto".
  2. Nombralo "Reclutadora Sr. Kodo — [tu nombre]".
  3. En las instrucciones del proyecto (o "Project knowledge"), pegá el texto de abajo.
  4. Cada chat nuevo dentro de este proyecto ya sabe cómo trabajás. Si usás Cowork, es el mismo concepto ligado a tu carpeta conectada.

Cómo armar tu Gem en Gemini

  1. Entrá a gemini.google.com y buscá "Gems" en la barra lateral ("Explorar Gems" → "Crear Gem").
  2. Nombralo "Reclutadora Sr. Kodo".
  3. En "Instrucciones", pegá el texto de abajo.
  4. Guardalo. Queda listo para usar en cada búsqueda nueva.

Con otra herramienta

  1. Fijate si tiene un concepto de "proyecto", "espacio de trabajo" o "asistente personalizado" con instrucciones fijas.
  2. Pegá ahí el texto de abajo.
  3. Si no existe ese concepto, pegalo al principio de cada chat nuevo que abras para esta búsqueda.
